# uhhwhydoyoudothatvisualworkbench | [![][cf-shield]][cf-link] [![][mr-shield]][mr-link]

### This is a client only Neoforge mod that fixes an interaction between EMI, TMRV and Visual Workbench. E.g if you need 4 items to craft something but 2 of them are already in the crafting table, adding the recipe to the table via EMI doesn't work because it only sees the 2 items in your inventory. This mod simply fixes this by disabling the Visual Workbench JEI plugin and letting EMI handle that.
